# Summary

Version 4 of the cable gland consists in a split in half bolt design with alignment notches, inside grip texture and a mounting nut. The default (and STL) size is 8 mm inner diameter and 12 mm outer diameter. Can be customized with OpenSCAD. The main advantages of this version are:

- Simple, robust and scalable design
- Fully customizable in OpenSCAD
- Complete 3d printed solution, no need of additional hardware parts
- Easy to print as is, without supports
- Easy to mount thanks to the split in half design, alignment notches and mounting nut
- Inside texture provides better grip
