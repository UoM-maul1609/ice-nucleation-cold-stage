# Summary

Tiny cable clip (1x2cm) with a flat base, useful for holding USB cables to the table. Meant to be used in combination with double sided tape (e.g. 3M VHB), not screws. Needs to be printed laying on it's side, otherwise it will break from layer separation when flexed. Use 100% infill.
